# 前端路由

前端路由实现起来其实很简单，本质就是监听 URL 的变化，然后匹配路由规则，显示相应的页面，并且无须刷新页面。

## hash 模式

### 什么是 hash ?

hash 是指 URL 当中 # 字符后面的那些字符。

```JavaScript
// 可以 hashchange 事件监听到 hash 值的变化
window.addEventListener('hashchange', (e) => {
    // 可以通过 location.hash 获取到 hash值
    console.log(location.hash)
    console.log(e);
});
```
